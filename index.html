HAW Events

"HAW Events" is an app developed to display events happening on the Finkenau Campus of the University of Applied Sciences. Users are able to view and filter all events, as a list and as a calender view. Users are also able to favorite events and thereby save them to a favorite site and view them even offline. THe app is connected with an API to a MongoDB Database, which resides on the university server. Additionally there is a website with a form to add events to the database. To add events one has to have a login which only selected staff members received. My main task was the app development, including intigrating designs and functionalities. I taught myself how to use the Flutter framework and its programming language Dart.
